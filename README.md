# IKL – Inter‑Kations‑Linie

IKL ist das Modul, das die tiefste Beziehungsschicht des iki1uc‑Systems beschreibt.  
Es verbindet die Operatoren, Nutzer, Ziele und Bewegungen zu einer einzigen Linie,  
die am Ende **REAL** erzeugt.

---

## 🔹 Zweck

IKL berechnet:

- Beziehungen  
- Interaktionen  
- Ketten  
- Bedeutungen  
- System‑Realität  

IKL ist die Schicht unter:

- 9hoch9  
- 81_81_1  
- META‑CORE  
- AXI  
- MXU  
- RESPO‑neu  

IKL ist die **Realitäts‑Ebene**.
